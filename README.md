# Ganesh AI Representative

AI-powered personal representative chatbot built using RAG.

## Features
- Answers questions about Ganesh's skills and projects
- Uses ChromaDB for vector storage
- Uses Sentence Transformers for embeddings
- Uses OpenRouter LLM for responses
- Streamlit UI

## Tech Stack
- Python
- Streamlit
- ChromaDB
- Sentence Transformers
- OpenRouter

## Run Locally

pip install -r requirements.txt

python ingest.py

streamlit run app.py
